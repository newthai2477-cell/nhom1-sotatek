# Mộc Cafe ☕

Landing page giới thiệu thương hiệu cà phê rang xay thủ công — xây dựng bằng HTML & CSS thuần, không JavaScript.

## Cấu trúc

```
├── index.html
└── style.css
```

## Các section

| Section | Nội dung |
|---|---|
| Header | Logo + điều hướng cố định |
| Hero | Tagline thương hiệu + minh hoạ cốc cà phê CSS |
| About | Câu chuyện & triết lý pha chế |
| Menu | 6 thức uống nổi bật |
| Sản phẩm | Hạt rang & dụng cụ mang về |
| Footer | Địa chỉ, giờ mở cửa, liên hệ |

## Chạy

Mở `index.html` trực tiếp trên trình duyệt
